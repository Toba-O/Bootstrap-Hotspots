Bootstrap element hotspots

Getting started

To get started include Bootstraps styling as wel as the Bootstrap jquery libraries to your project. For more information on how to do this go to Bootstraps documentation.
After you've done that add the bootstrap-hotspot.css file to your page and you're ready to go!

Adding a hotspot is easy!

Create an element like the title above and add the hotspot element <span class="highlight-spot top left" data-toggle="popover" data-content="And here's some amazing content. It's very engaging. Right?"></span>. The hotspot uses Bootstraps builtin popovers. So there's no need for any extra javascript libraries. The hotspots styling uses position absolute so make sure the element you add it to is positioned relative.

Positioning the hotspot!

As mentioned before the hotspots are positioned absolute. There are six classes included in the styling to position the element. The default positioning classes are: top, bottom, left and right. This will position the element at the top or bottom and left or right. You should add either top or bottom as well as either left or right. You can center the hotspot vertically by using the center-y class, or horizontally by using the center-x class. If you don't add classes the hotspot will default be at the top left of the element. Make sure the element you add it to is positioned relative by either adding element{position:relative} to the css of the element or by using bootstraps .position-relative class.

Adding hotspots to elements without children!

Certain HTML-elements do no allow you to add a child element to them. This means you can't add the hotspot element to elements like inputs and images. But you can still add a hotspot to those elements by adding a parent wrapper element. For example add a div with class position-relative and then add the element and the hotspot as seperate children.

Adding hotspots to buttons or links!

Watch out. If you add the hotspot to a button or link, clicking it will cause the button or link to get clicked. For these situations you should also use a parent wrapper and add the highlight and the button or link as seperate children.

It's as simple as that!

There's nothing more to it. You just add the hotspot element with Bootstraps default popovers. The text you want to show in the hotspot you add to the data-content attribute of the hotspot. You can also add data-title to add a title to the hotspot as well as use Bootstraps popover data-placement attribute to determine if the text should be shown above, below, to the left or the the right of the hotspot.
More info about Bootstraps popovers can be found here!
